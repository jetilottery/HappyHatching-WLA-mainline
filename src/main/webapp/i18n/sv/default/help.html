Happy Hatching

Kommer Happy Hatchings höna kläcka rätt nummer och ge dig stora vinster? Välj kycklingar eller hönor i jakten på matchande nummer! Vissa kycklingar ger direktvinster medan andra ger multiplikatorer för ännu större vinster!

Så här satsar du

Lottkostnad

Visar den aktuella lottkostnaden.

Tryck på vänsterpilen (–) för att minska lottkostnaden.

Tryck på högerpilen (+) för att öka lottkostnaden.

Tryck på KÖP för att dra lottkostnaden från saldot och spela.

Så här spelar du

Målet med Happy Hatching är att matcha DINA NUMMER med LYCKONUMMER och/eller att hitta direktvinstkycklingarna med 1x, 2x eller 5x på magen.

Välj varje höna för att avslöja ett LYCKONUMMER. Välj varje ägg för att avslöja DINA NUMMER och vinstvärden.

När ett av DINA NUMMER matchar ett LYCKONUMMER, vinner du DITT NUMMERS vinstvärde.

Hitta en direktvinstkyckling för att vinna vinstvärdet direkt.

Hitta en 2x-kyckling för att vinna dubbla vinstvärdet.

Hitta en 5x-kyckling för att vinna fem gånger vinstvärdet.

För att slutföra spelet väljer du alla ägg och hönor eller trycker på AVSLÖJA ALLA. Därefter dyker ett meddelande upp som visar spelets utfall, varefter spelet avslutas.

Avslöja alla

Tryck på AVSLÖJA ALLA för att automatiskt spela klart spelet. Spelet fortsätter tills alla val har gjorts.

Tryck på STOPP för att fortsätta spelet i manuellt spelläge.

Regler

Missbruk eller funktionsfel ogiltigförklarar alla utbetalningar och spel.

Direktvinster läggs till det totala vinstvärdet.

Vinster visas i vald valuta.

Val som gjorts av spelare under spelets gång, påverkar inte spelets utfall.
