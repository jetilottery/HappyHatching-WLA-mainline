Happy Hatching

Les poules de Happy Hatching tricoteront-elles les bons numéros pour vous faire gagner gros? Pour le savoir, appuyez sur les poussins et les poules pour trouver des numéros identiques! À l'éclosion, certains poussins affichent un lot instantané et d'autres, un multiplicateur qui permet de gagner encore plus!

Comment miser

Coût du billet

Affiche le coût actuel du billet.

Appuyez sur la flèche gauche (-) pour diminuer le coût du billet.

Appuyez sur la flèche droite (+) pour augmenter le coût du billet.

Appuyez sur ACHETER pour débiter votre solde du prix d’un billet et commencer la partie.

Comment jouer

Dans le jeu Happy Hatching, vous gagnez lorsque certains de VOS NUMÉROS sont identiques aux NUMÉROS CHANCEUX ou lorsque vous trouvez des poussins offrant des lots instantanés multipliés par 1, 2 ou 5.

Appuyez sur chaque poule pour faire apparaître un NUMÉRO CHANCEUX. Appuyez sur chaque œuf pour faire apparaître VOS NUMÉROS et les lots correspondants.

Si VOTRE NUMÉRO est identique à un NUMÉRO CHANCEUX, vous gagnez le lot associé à VOTRE NUMÉRO.

Trouvez un poussin offrant un lot instantané et gagnez instantanément le lot correspondant.

Trouver un poussin avec un multiplicateur de 2 pour gagner le double du lot affiché.

Trouver un poussin avec un multiplicateur de 5 pour gagner cinq fois le lot affiché.

Pour compléter le jeu, sélectionnez tous les œufs et les poules ou cliquez sur TOUT AFFICHER; le jeu affiche un message indiquant le résultat de la partie, et la partie prend fin.

Tout afficher

Appuyez sur TOUT AFFICHER pour que la partie se joue automatiquement. La partie continue jusqu’à ce que toutes les sélections aient été effectuées.

Appuyez sur ARRÊTER pour continuer la partie en mode manuel.

Règles

Toute utilisation inappropriée et toute défectuosité annulent jeux et paiements.

Les lots instantanés sont ajoutés aux gains totaux.

Les gains sont affichés en argent.

Les choix effectués par le joueur pendant le jeu n’ont aucune incidence sur le résultat de la partie.
